1. Download Gitbash Terlebih Dahulu!

Masuk ke situs "git-scm.com" lalu tekan download dan pilih OS yang sesuai dengan perangkatmu. Disini saya menggunakan windows 64-bit, jadi saya download versi yang untuk windows.

Install dan tekan Next, biarkan settingan default saja, tidak perlu di ubah.

2. Buka Command Prompt

Buka command prompt dan ketik "git" lalu tekan Enter.

Jika muncul tulisan-tulisan seperti ini, berarti kamu telah berhasil menginstall Git di perangkatmu.

3. Buat Repository Baru Di GitHub

Setelah itu, copy link repository github kamu.

Tekan tombol bagian paling kanan seperti di gambar.

4. Buka Projectmu yang Ingin Kamu Upload

Buka terminal dengan tekan tombol "CTRL + `" dan pilih Git Bash

5. Masukkan Sintaks - Sintaks Git


Masukkan sintaks-sintaks Git di terminal yang kamu buka tadi seperti gambar di bawah ini.

*Letakkan link repository yang sudah kamu Copy ke sintaksnya.

  • git init
  • git add .
  • git commit "nama upload"
  • git branch -M master
  • git remote add origin "link repository kamu"
  • git push -u origin master

SELAMAT! Kamu telah berhasil upload project kamu ke Github.
